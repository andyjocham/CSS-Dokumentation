Verwendung von CSS:

inline: Option (style) zum Tag
CSS-Formatierung gilt nur für diesen Tag an dieser Stelle
Datei-bezogen: CSS-Angaben im head-Tag
CSS-Formatierung gilt für jeden Tag in der Datei
externe Datei: eigene CSS-Datei
CSS-Formatierung erfolgt in einer eigenen Datei und kann somit in mehreren HTML-Seiten eingebunden werden

Syntax von CSS-Angaben

 allgemein gilt:
  • case-sensitive
  • Vererbung der Angaben
  • Einrückungen bzw. Leerstellen haben keine Auswirkung
  • Kommentare (mehrzeilig) mit /* und */

Wert-Angaben:
  • Größen: in Prozent (75%), Ganzzahl/Fließkommazahl + Einheit
  • Einheiten: cm, mm, in (inch), pt (Typographie-Punkt), pc (Pica-Punkt), px (Pixel), em (Laufweite von "m"), ex (Höhe von "x")
  • Farbangaben: vordefinierte Bezeichnung (amerikanisch)
  • RGB-Code (hexadezimal): 3-stellig oder 6-stellig
  • Werte von 00 bis ff
  • #4488aa --> hex 44 rot, hex 88 grün, hex aa blau
  • #48a --> hex 44 rot, hex 88 grün, hex aa blau
  • RGB-Code (dezimal): Werte von 0 bis 255
  • rgb(120, 80, 240) --> rot 120, grün 80, blau 240

verschiedene CSS-Angaben:

 Margin- bzw. Padding-Angaben:
  • eine Angabe: "oben/rechts/unten/links"
  • zwei Angaben: "oben/unten" "rechts/links"
  • drei Angaben: "oben" "rechts/links" "unten"
  • vier Angaben: "oben" "rechts" "unten" "links"
Text-align-Abgaben:
  • left (linksbündig), center (zentriert), right (rechtsbündig), justify (Blocksatz)
Border-Angabe:
  • border: Linien-Dicke Linien-Art Linien-Farbe
  • border-top: Linien-Dicke Linien-Art Linien-Farbe /* oben */
  • border-right: Linien-Dicke Linien-Art Linien-Farbe /* rechts */
  • border-bottom: Linien-Dicke Linien-Art Linien-Farbe /* unten */
  • border-left: Linien-Dicke Linien-Art Linien-Farbe /* links */
Linien-Art:
  • solid - durchgezogen
  • double - Doppelinie durchgezogen
  • dashed - gestrichelt
  • dotted - gepunktet
  • ridge - erhaben
  • groove - abgesenkt
  • inset - eingedrückt
  • outset - herausgedrückt

Links optisch verändern:

 Standard-Optik fuer alle Links Link-Zustände anpassen Standard-Optik fuer alle Button-Links
Zustände von Links:
  • link : nicht benutzter Link
  • visited : Link wurde bereits benutzt, Seite wurde besucht
  • focus : Link ist angewählt (Tab-Taste)
  • hover : Maus schwebt über dem link
  • active : gedrückter Link
